# Data Cleaning & Visualization Project using Python

##  Project Overview

This project demonstrates the complete data analysis workflow using a Netflix dataset. The primary objective is to clean raw data, preprocess it, and create meaningful visualizations to uncover patterns and insights.

The project covers the essential stages of data analysis, including data cleaning, exploratory data analysis (EDA), and data visualization using Python libraries.

---

##  Objectives

- Load and inspect the raw dataset
- Clean and preprocess the data
- Handle missing values
- Remove duplicate records
- Remove unnecessary columns
- Perform exploratory data analysis (EDA)
- Visualize important trends and patterns
- Draw meaningful conclusions from the data

---

##  Technologies Used


